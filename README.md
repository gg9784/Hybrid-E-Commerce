# 🛒 Hybrid MERN E-Commerce Platform

A full-stack **hybrid e-commerce web application** built using the MERN stack, combining traditional online shopping with **real-time video calling and chat** between users and sellers for an enhanced interactive experience.

---

## 🚀 Features

### 🛍️ E-Commerce Functionality
- Browse products with categories and search
- Add to cart & wishlist
- Secure checkout process
- Order history & tracking
- Email confirmation on successful orders

### 🔐 Authentication & Authorization
- JWT-based authentication
- Role-based access (User / Admin)
- Protected routes and APIs

### 🎥 Real-Time Communication (Hybrid Feature)
- Peer-to-peer **video calling** using WebRTC
- Real-time messaging with **Socket.IO**
- Live interaction between buyers and sellers

### 🧑‍💼 Admin Dashboard
- Add / update / delete products
- Manage inventory
- View orders and analytics

---

## 🧱 Tech Stack

### Frontend
- React.js
- Redux Toolkit
- Tailwind CSS / CSS

### Backend
- Node.js
- Express.js
- MongoDB

### Real-Time & Communication
- WebRTC (Video Calling)
- Socket.IO (Signaling + Chat)
